## Description

This a proof of concept to start using web sockets with node and socket.io.

Project from React + Socket.io course by https://github.com/Klerith

## Technologies

- Node
- TypeScript
- Socket.io

## Installation

`npm i`

## Setup

Create `.env` file
```

PORT=3001

```

## Run project

`npm start`
